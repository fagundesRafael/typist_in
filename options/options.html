Oitiva Assistente - Configurações

Para utilizar esta extensão, você precisa configurar sua chave API do Google Gemini. A chave API é necessária para gerar perguntas com base no contexto da ocorrência policial.

Sua chave API é armazenada apenas localmente no seu navegador e não é compartilhada com ninguém.

Como obter uma chave API do Google Gemini

  1. Acesse o Google AI Studio (você precisará fazer login com sua conta Google).
  2. Clique em "Get API key" ou "Obter chave API".
  3. Se você já tem uma chave API, ela será exibida. Caso contrário, clique em "Create API key" ou "Criar chave API".
  4. Dê um nome para sua chave API (por exemplo, "Oitiva Assistente").
  5. Copie a chave API gerada e cole-a no campo abaixo.

Importante: A chave API do Google Gemini é gratuita para uso pessoal, mas tem limites de uso. Consulte a documentação de preços para mais informações.

Mantenha sua chave API segura e não a compartilhe com outras pessoas.
